> We are currently testing the Avo Public API and it's not yet in general availability. Reach out to us for access.

The Avo API is organized around REST, has predictable resource-oriented URLs, which are versioned by endpoint.
Avo API returns JSON encoded responses and uses standard HTTP response codes.

Avo API Uses <Link href="https://en.wikipedia.org/wiki/Basic_access_authentication"><a>Basic Authentication</a></Link> to authenticate requests.
